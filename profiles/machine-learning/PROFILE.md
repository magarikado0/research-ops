# Machine Learning research profile

機械学習モデルの構築と評価を扱う。
汎用コアの「研究材料」はデータセット、「活動」は学習・評価・分析のrunとして具体化する。

## PANELの失敗モード

- **査読者2号**：予測タスクと手法に新規性・意義があるか。既存研究で解決済みではないか。
- **データ懐疑派**：リーク、ラベル品質、標本選択、分布シフトによって評価が過大になっていないか。
- **ベースライン原理主義者**：線形、直近値、規則などの単純手法を十分に上回っているか。
- **利用担当**：予測を誰が何に使うか。指標と運用上の損失が一致しているか。
- **撤退判断担当**：改善幅、計算費用、データ制約を踏まえ、研究を止める条件は何か。

## 根拠の例

run ID、データ版、分割定義、評価コード、指標、アブレーション、論文、実装箇所を使う。

# Changelog

## 2.0.0 — 2026-07-16

- Codexアダプタを非推奨のカスタムプロンプトから `.agents/skills/` へ移行。
- ログ層から完全版を作るDOCS SYNCと、文書層から文脈パックを作るSTATE SYNCを分離。
- 両同期をコミットSHAと時刻による差分取得とし、DOCS SYNCには活動ログカーソルを追加。
- 両同期の既定をdry-run化。
- 分野固有の前提を汎用コアから分離し、5種類の研究プロファイルを追加。
- 実験ログを汎用的な活動ログへ拡張し、場所と形式を `research-ops.yml` で宣言する仕様を追加。
- 対話形式と非対話形式に対応した安全なインストーラを追加。
- 一つの文書層へ複数の物理パスを登録できる `document_roots` を追加。
- AUDITに独立コンテクストを必須化。
- PANELの審議記録と、人間が承認した正式決定を分離。
- 最小導入時に存在しないSTATEファイルが目次へ残る問題を修正。
- 検証スクリプトとCIを追加。
